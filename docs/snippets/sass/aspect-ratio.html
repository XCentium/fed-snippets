# Aspect Ratio

by Andrew Aponte

Resizes and crops an image according to the specified aspect ratio.


SCSS

@mixin aspect-ratio($width, $height) {
    position: relative;
    height: 0;
    padding-bottom: #{ ($height / $width) * 100 } + "%";

    img {
        position: absolute;
        width: 100%;
        height: 100%;
        object-fit: cover;
    }
}

.image {
    @include aspect-ratio(16,9);
}

HTML

<figure class="image">
    <img src="path/to/img.jpg" />
</figure>
